PayPay
======

2023-11-08 08:09

PayPay是日本最主流、最受欢迎的数字钱包，几乎覆盖了日本一半的人口。

----

| **支付流程** | 重定向 | **自动扣款** | ✔️ |
| **买家国家/地区** | 日本 | **退款** | ✔️ |
