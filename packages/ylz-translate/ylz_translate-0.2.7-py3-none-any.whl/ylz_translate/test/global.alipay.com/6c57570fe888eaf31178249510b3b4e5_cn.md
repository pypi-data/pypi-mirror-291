支付方式类型
====================  
2024-05-11 10:13  
蚂蚁通为您提供标准的集成解决方案，以便您集成各种支付方式。此解决方案节省了您未来集成支付方式和扩展功能的开发工作。对于某些支付方式在获取过程中可能具有的特殊要求，您只需在集成时关注这些支付方式在API参数上的差异。  

支付方式类型
====================  
蚂蚁通提供了来自不同地区的多种支付方式类型。每种支付方式都提供相似的用户体验。您可以根据需要集成以下支付方式类型：  
*   [支付宝+支付方式](#mt4eB)
*   [钱包](#IK789)
*   [移动银行应用](#KSRbI)
*   [银行转账](#bR1hN_zh)
*   [在线银行](#llC45_zh)
*   [银行卡](#DDFip_zh)
*   [场外交易](#FwduU_zh)
*   [先买后付](#A06j5_zh)
您可以遵循Antom提供的标准集成解决方案，轻松集成这些支付方式。
