泰国汇丰银行
--------------

2023-11-08 08:09

泰国汇丰银行（KTB）是一家国有上市银行，也是泰国第二大银行。KTB 还是大多数政府机构的首选银行。

KTB 拥有超过 1500 万泰国用户。目前，银行为用户提供三种支付方式：网上银行、移动银行应用和通过线下 ATM 进行的银行转账。
