注册
============

2023年7月14日 10:00  
POST /v1/merchants/registration  
使用**注册** API 向支付宝注册二级商户。

结构
========

消息由头和体组成。以下部分关注体的结构。关于头的结构，请参阅：

*   [请求头](https://global.alipay.com/docs/ac/ams/api_fund#ML5ur)
*   [响应头](https://global.alipay.com/docs/ac/ams/api_fund#WWH90)

**注意**：将每个字段（除数组外）的数据类型设置为字符串。这意味着必须使用双引号（" "）包围字段值。例如：

*   如果字段的数据类型为Integer，其值为20，应设置为"20"。
*   如果字段的数据类型为Boolean，其值为true，应设置为"true"。
### 请求参数  
显示全部  
#### registrationRequestId 字符串  必填  
用于识别注册请求的唯一ID。  
关于此字段的更多信息：  
*   此字段是一个API幂等字段。商家使用registrationRequestId字段进行幂等控制。对于使用相同registrationRequestId值发起并达到最终状态（S或F）的注册请求，应返回相同的结果。
*   最大长度：64个字符  
#### registrationNotifyURL URL  
商家提供的用于接收注册通知的URL。  
关于此字段的更多信息：  
*   最大长度：2048个字符  
#### passThroughInfo 字符串  
仅用于站点透明数据传输的键值结构。  
关于此字段的更多信息：  
*   最大长度：2048个字符  
#### merchantInfo MerchantRegistrationInfo 对象  必填  
商家信息  
显示子参数  
#### productCodes 字符串数组  必填  
产品代码。对于店内支付，此字段的有效值为 `IN_STORE_PAYMENT`。  
#### storeInfo StoreRegistrationInfo 对象  
商店信息  
注意：当 _productCodes_ 为 `IN_STORE_PAYMENT` 时，此字段才必需。如果没有传递 _storeInfo_，将返回 `PARAM_ILLEGAL` 错误。  
显示子参数
### 响应参数  
显示全部  
#### result 结果对象 **必需**  
请求结果，包含状态和错误代码等信息。  
显示子参数  
#### passThroughInfo 字符串  
仅用于站点间透明数据传输的键值结构。  
关于此字段的更多信息  
*   最大长度：2048 个字符

注意:  
对于主要参数的重要细节，请参阅以下列表：  
*   对于二级商户注册请求，_merchantId_ 和 _referanceMerchantId_ 必须全局唯一。如果注册重复，将返回 `REPEAT_REGISTRATION` 错误。
*   如果 _field_ 的类型为 _Enum_，则 _field_ 的值必须在指定的 _Enum_ 范围内。例如，_attachmentType_、_contactType_、_registrationType_、_businessType_ 和 _productCodes_ 是枚举类型。如果传递的值超出 _Enum_ 范围，将返回 `PARAM_ILLEGAL` 错误。
*   如果未传递必填字段，将返回 `PARAM_ILLEGAL` 错误。
* 对于字符串类型的字段，如果参数长度超过指定的最大长度或小于最小长度，会返回`PARAM_ILLEGAL`错误。
* _website_的`url`必须是有效的URL，否则会返回`PARAM_ILLEGAL`错误。
* _address_必须是Google Maps可以识别的地址。
* _merchantInfo.registrationDetail.attachments.file_必须是一个可以下载文件的URL。
* 对于_merchantInfo.merchantMCC_的值，详情请参考[MCC代码](https://global.alipay.com/docs/ac/ref/mcccodes)。
### 结果/错误代码  
| 代码 | 值 | 消息 | 进一步操作 |
| --- | --- | --- | --- |
| SUCCESS | S | 成功 | 注册成功，无需进一步操作。 |
| INVALID\_CLIENT | F | 客户端无效。 | 检查\_clientId\ 是否正确。 |
| MEDIA\_TYPE\_NOT\_ACCEPTABLE | F | 服务器不支持客户端可接受的媒体类型。 | 检查媒体类型是否正确。 |
| METHOD\_NOT\_SUPPORTED | F | 服务器不支持请求的HTTP方法。 | 检查HTTP方法是否正确。 |
| REPEAT\_REQ\_INCONSISTENT | F | 重复请求不一致。 | 更改\_registrationRequestId\ 并重新调用接口。 |
| REGISTRATION\_UNDER\_REVIEW | F | 之前提交的注册正在审核中。不允许新的注册或修改。 | 提交的注册正在审核中。请等待最终结果。 |
| REPEAT\_REGISTRATION | F | 商户重复注册 | 提示用户不要重复注册。 |
