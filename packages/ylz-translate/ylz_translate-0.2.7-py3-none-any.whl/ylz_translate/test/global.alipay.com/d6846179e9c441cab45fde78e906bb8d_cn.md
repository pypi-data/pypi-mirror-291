bKash
=====

2023-03-09 10:53

bKash Limited（bKash）是孟加拉国一家由银行主导的移动金融服务提供商，根据中央银行（孟加拉国银行）的许可和批准，作为BRAC Bank Limited的子公司运营。bKash通过移动电话为孟加拉国的银行服务未覆盖和已覆盖人群提供安全、便捷和简单的支付及转账服务。
