*   如果\_declarationAmount\_
    发生变化，新的申报金额不能超过原始的总交易金额。

**重传响应**

以下列出了不同情况下的响应：

*   如果不满足重传条件且没有任何参数值改变，返回幂等成功的响应。
*   如果不满足重传条件但有一个或多个参数值已改变，将返回错误代码`CONTEXT_INCONSISTENT`或其他错误代码。
*   如果满足重传条件且重传成功，返回值将与第一次成功传输时相同。
